# My WC cli tool

## Description
This is a simple cli tool that can be used to count the number of words, lines and characters in a file.

## Usage
```bash
python my_wc.py <file>
```

## Example

### Bytes count
```bash
python my_wc.py test.txt

python my_wc.py -c test.txt
```

### Lines count
```bash
python my_wc.py -l test.txt
```
